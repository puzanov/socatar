Socatar

Web service to use profile photos in your page.

Just use link in your HTML:
<img src="https://socatar.com/twitter/yegor256" />

Currently supported sources:
  • Facebook with id, like 100001584476227: /facebook/1531296526
  • GitHub with login, like terales: /github/yegor256
  • Google with Google+ id, like +SriramSaroop: /google/+SriramSaroop
  • Gravatar with email: /gravatar/email@example.com
  • Twitter with username, like robwormald: /twitter/yegor256

Images will load fast thanks to CDN and browser cache

All images are cached in Cloudflare for about seven days and cached in browser for a year, so after the first load they will load very fast. To update image cache just add anything to query: /facebook/1531296526?v=2
